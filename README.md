# czh-quizzes

A React-based quiz experience for Chinese Zero to Hero course materials.

## Development
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```

## Production build
Create an optimized build ready for static hosting:
```bash
npm run build
```
The output is generated in the `dist/` directory.

## Preview build
Serve the production bundle locally to verify the deployment artifacts:
```bash
npm run preview
```
